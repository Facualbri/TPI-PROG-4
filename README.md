# ProdeUTN

## Descripción
Sistema de pronósticos deportivos desarrollado en Java y Spring Boot.

## Funcionalidades
- Registro y login con JWT
- Gestión de equipos
- Gestión de fechas
- Gestión de partidos
- Pronósticos deportivos
- Ranking global
- Grupos privados

## Tecnologías
- Java 17
- Spring Boot 3
- PostgreSQL
- Spring Security
- JWT
- JPA/Hibernate
